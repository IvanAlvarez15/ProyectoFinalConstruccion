Door-of-drums API

Introduction

Welcome to the Door-of-drums API. You can use our API to access Door-of-drums API endpoints, which can get information on the songs that are used in the game, the user info and the records of the users.

User

Get user info

This endpoint retrieves all the info from all users.

HTTP Request

GET http://doors-of-drums.games:8080/api/users.json

Result of this query

Unity user info

Get the data from the server to the game.

This endpoint retrieves the information of the player from the web so they can access the game.

HTTP Request

POST http://doors-of-drums.games:8080/api/private/

Data Params

You need to be register in our web so you can access the game.

Result of this query

{
        "username": "LoboSalvaje",
        "status": "1",    
      }

Deleate the user

Get the data of an specific user.

This endpoint redirects to a confirmation page so the user can deleate their info.

HTTP Request

POST http://doors-of-drums.games:8080/Borrar/id

Where id is the id of the user

Data Params

To use this endpoint you need to be logged in in the website.

Result of this query

This will redirect to the a page of confirmation where the user can delete their info.

Username and hashed password for unity

Get the info of the user so they that access the game.

HTTP Request

POST http://doors-of-drums.games:8080/LoginUnity/

Data Params

You need to be in the game and register so the endpoint can be use.

Result of this query

This will redirect to the the menu scene of the game.

Deleate user info in unity

Get the info of the user so it can be deleated in unity.

HTTP Request

POST http://doors-of-drums.games:8080/BorrarUnity/

Data Params

You need to be in the game and register so the endpoint can be use.

Result of this query

This will deleate all the user info in our database.
